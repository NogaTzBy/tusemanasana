# Alcance del MVP (v1.0)

## Incluye (Scope v1.0)
- **Frontend & Onboarding:** Landing page, cuestionario de 6 preguntas, resumen personalizado post-cuestionario simulando IA, testimonios, 1 receta gratuita.
- **Suscripciones (Checkout):** Integración con Stripe para Trial de 3 días (plan mensual y anual) y cobro automático al día 3.
- **Core App:** Dashboard con plan semanal (7 días x 3 comidas). Vista de la semana y receta destacada.
- **Funcionalidad Especial:** Opción discreta de cambiar receta por día.
- **Exportables PDF:** Generación de PDF del plan semanal y lista de compras agrupada por categorías.
- **Admin Panel:** Dashboard con métricas clave y gestión básica de usuarios/suscripciones.

## No Incluye (Scope v2.0+)
- Aplicación mobile nativa (iOS / Android).
- Videos demostrativos de recetas.
- Artículos o blog de nutrición y bienestar.
- Integraciones con dispositivos de salud (Apple Watch, Oura, etc.).
- Programa de referidos o recompensas.
- Comunidad interna o foros entre usuarias.
- Soporte para múltiples idiomas.
